Lab 1: Intro to Logic Simulation

-----------
DESCRIPTION

This lab will introduce you to a schematic entry logic simulation program, Multimedia Logic (MMLogic or MML). In this lab you will practice creating truth tables and implementing logic based on those truth tables.

FILES

INSTRUCTIONS

Part A: We are connecting the wires from the user input switches to the 7 segment display component.
Part B: I find out the regular in the truth table and use xor to connect the display.
Part C: I use SOP to follow the truth table and use nand and nor gate to build the same thing.
